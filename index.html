For Console Application:

open console

Add todo: app.ins(model.$root, [todo text: String], [todo parent: String]);

app.ins('add this todo', 'master watchandcode');

(Without a todo parent string, the app will add the todo to the root level.)

Delete todo: app.del(model.$root, [todo text: String]);

app.del('delete this todo');

Edit todo: app.edit([original todo text: String], [changed todo text: String]);

app.edit('edit this todo', 'this has been edited');

Toggle todo: app.tog([todo text: String]);

Toggle All: app.togall();
